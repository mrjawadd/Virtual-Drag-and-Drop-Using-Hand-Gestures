# Virtual Drag and Drop Using Hand Gestures

Computer vision based app that lets users drag objects using hand gestures.
Uses OpenCV and hand tracking to detect pinch gestures in real time.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python main.py
3. Press Q or ESC to exit.

## Note
This project requires a webcam.
